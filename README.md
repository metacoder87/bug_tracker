# README

This App is a bug tracking system meant to be used inhouse by GM Cube Designs employees, but will be available for outside use and demonstration purposes. 

This Bug Tracker is to improve production performance and speed by providing an easy go to source to find what bugs need to be addressed within a project before the next increment is shipped.

For more indepth understanding of the Bug Tracker please refer to the Software Requirement Specification PDF that is attached.
